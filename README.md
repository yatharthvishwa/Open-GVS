# Open-GVS
control a person with gvs!!
this can control your balance by applying a controlled current to your mastoids, this project uses diy 3d printed electrodes which you can apply to your mastoids which are connected to the pcb
I was inspired to make this project because of a open sauce video https://www.youtube.com/watch?v=9Om2X6QcTgw
